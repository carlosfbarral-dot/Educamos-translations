# Educamos Translations

## Project Documentation

### Features
- Multi-language support
- Easy to use interface for managing translations
- Integration with the Educamos platform

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/carlosfbarral-dot/Educamos-translations.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Educamos-translations
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Database Schema
- **users**: Stores user information such as name, email, and role.
- **translations**: Contains the translation keys, values, and the associated languages.
- **languages**: A list of supported languages including language codes.

### Supported Languages
- English (en)
- Spanish (es)
- French (fr)
- German (de)
